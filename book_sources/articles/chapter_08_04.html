Typedef, Enum, and Union

Typedef, Enum, and Union

1. 型態定義指令(typedef)

  • 型態定義指令可將原有的型態或結構以有意義的新名稱取代，提高程式的可讀性

    typedef 原型態 新定義型態;

    Example 1

    typedef int INTEGER;     // INTEGER被定義成int型態
typedef char STRING[20]; // STRING被定義成長度20的字元陣列
typedef struct hotel{
    INTEGER room_number;
    STRING hotel_name;
}HOTEL_INFO;             // HOTEL_INFO被定義成struct hotel型態

int main(){
    HOTEL_INFO myhotel = {920, "Guitar_hotel"}; // 不必加上struct

    cout << myhotel.room_number << endl;
    cout << myhotel.hotel_name << endl;

    return 0;
}

    Output 1

    Line 11: 920
Line 12: Guitar_hotel
2. 列舉型態指令(enum)

  • 列舉型態指令的功能是給予一組常數集合而成的列舉成員，各常數值不同的命名。若無指定數值，預設將第一個列舉成員設為0，後面列舉成員的常數值則依序遞增

    (declaration)
    enum 列舉型態名稱{
        列舉成員1 = 預設值1;
        列舉成員2 = 預設值2;
        ...
    };
    (definition)
    enum 列舉型態名稱 列舉型態變數;

    Example 2

    enum drink{
    coffee,
    milk,
    tea = 41,
    water,
    uice = 10
};

enum drink mydrink;

mydrink = coffee;
cout << mydrink << endl;

mydrink = milk;
cout << mydrink << endl;

mydrink = tea;
cout << mydrink << endl;

mydrink = water;
cout << mydrink << endl;

mydrink = juice;
cout << mydrink << endl;

    Output 2

    Line 12: 0
Line 14: 1
Line 16: 41
Line 18: 42
Line 20: 10
3. 聯合型態指令(union)

  • 聯合型態指令與結構型態指令在宣告、定義方法或是成員存取上都非常相似，不同的是結構中每個成員擁有各自的記憶體空間，而聯合卻是共用記憶體空間，並以最大長度記憶體的成員作為聯合的空間大小

    (declaration)
    union 聯合型態名稱{
        資料型態 資料成員1;
        資料型態 資料成員2;
        ...
    };
    (definition)
    union 聯合型態名稱 聯合變數;

    Example 3

    int encode(int plain_text);
int decode(int cipher_text);

int main(){

    int plain_text = 1234;

    int cipher_text = encode(plain_text);
    int decoded_text = decode(cipher_text);

    cout << cipher_text << endl;
    cout << decoded_text << endl;

    return 0;
}
int encode(int plain_text){
    union{
        int num;
        char c[sizeof(int)];
    }u1;

    u1.num = plain_text;

    for(int i = 0; i < sizeof(int); i++){
        u1.c[i] += 32;
    }

    return u1.num;
}
int decode(int cipher_text){
    union{
        int num;
        char c[sizeof(int)];
    }u1;

    u1.num = cipher_text;

    for(int i = 0; i < sizeof(int); i++){
        u1.c[i] -= 32;
    }

    return u1.num;
}

    Output 3

    Line 11: 538977522
Line 12: 1234
