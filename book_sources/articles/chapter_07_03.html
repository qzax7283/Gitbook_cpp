Variable Type Modifiers

Variable Type Modifiers

1. 自動變數
  • 必須宣告在函數區塊內，是系統預設的變數型態，在程式執行到區域範圍時才會被建立，當程式離開此範圍後所占用的記憶體空間會被釋放

    auto 資料型態 變數名稱;

2. 靜態區域變數
  • 宣告在函數區塊內，當函數執行完畢後，記憶體位址不會被清除，若再次呼叫該函數時，會將放在記憶體中的值取出，而不是定義的初值。若無設定初始值，系統會設定為0

    static 資料型態 變數名稱;

3. 外部變數

  • 外部變數是指宣告在函數區塊外的變數，也就是全域變數，會占用固定的記憶體空間。若無設定初始值，系統會設定為0。定義後以下所有的函數都可使用該變數，待程式執行結束後才將記憶體釋放。

  • extern修飾詞的功能可將宣告在函數區塊後方的外部變數引用到函數內使用

    extern 資料型態 變數名稱;

4. 靜態外部變數
  • 將靜態變數宣告在函數區塊之外即為靜態外部變數，與外部變數不同的是靜態外部變數只限於在同一個程式檔案使用
5. 暫存器變數
  • 使用CPU的暫存器來儲存變數，加快變數存取的效率

    register 資料型態 變數名稱;

Example 1

#include <iostream>
#include "Example 2.cpp"

using namespace std;

int func2(int a);

int x; // 全域變數

int main(){

    int a = 41; // 自動變數

    func1();
    cout << x << endl;
    cout << func2(a) << endl;
    cout << func2(a) << endl;

    return 0;
}
int func2(int a){
    static int sum = 10; // 靜態區域變數

    sum += a;
    return sum;
}

Example 2

#include <iostream>

using namespace std;

extern int x; // 外部變數

void func1(){
    x = 1;
}
